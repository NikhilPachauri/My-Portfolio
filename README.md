# My-Portfolio
A modern, responsive portfolio website built with pure HTML, CSS, and JavaScript. Includes dark mode toggle, smooth navigation, skills, projects, experience, education, and contact form, optimized for all devices.

Project Description A fully responsive, single-page portfolio website built using only HTML, CSS, and JavaScript. It showcases personal information, skills, projects, experience, and education with a modern UI, smooth animations, and an interactive dark mode toggle. The layout is optimized for all screen sizes without using any external libraries or APIs.

Features Fully responsive layout for mobile, tablet, and desktop

Modern hero section with call-to-action buttons

Skills section grouped by technology and professional strengths

Projects section highlighting tech stacks and responsibilities

Experience and education sections with clear timelines

Dark/light mode toggle with smooth transitions

Sticky navigation bar with smooth scrolling

Mobile-friendly hamburger menu

Contact section with basic form validation

Clean, semantic HTML structure and accessible design

Tech Stack HTML5 for structure

CSS3 (Flexbox, Grid, transitions) for styling and responsiveness

Vanilla JavaScript for interactivity and theme toggling
